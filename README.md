## Important domain setting

The canonical URLs and sitemap use:

`https://darehbaghipsychiatry.com`

If your final live domain is different, update `site_url` references inside:

- `index.html`
- `about.html`
- `robots.txt`
- `sitemap.xml`

## Button alignment
